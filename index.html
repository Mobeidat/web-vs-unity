Unity VS
Web App - PWA

Inroduction


Unity is platform for making real-time 3D projects for Games, Animation, Film, Transportation, Architecture, Engineering, Manufacturing & Construction provided as a native, desktop application for Windows, Mac and Linux.



PWA is a type of application software delivered through the web, built using common web technologies including HTML, CSS and JavaScript that runs in any browser on any operating system.

The pros and cons

Unity

Pros:
  • Platform support - Build once, deploy whenever and wherever
  • Insane graphics for interactive 3D experiences
  • Great for rendering 2D and 3D scenes
  • Asset store to save your time
  • Versatile Software


Cons:
  • License cost
  • Heavy and takes time to start up
  • Many features locked behind Pro (paid) version

PWA - web app

Pros:
  • Multi Platform Compatibility
  • Low Cost to Build
  • Faster UI
  • No Installation process
  • Searchable - can be indexed by search engines


Cons:

cons are dependent on the project itself

  • Not sutiable for complex 3D experiences
  • Too slow if using 3D models
  • Unrealiable if used for complex 3D
  • It may crash clients browser also if used for complex 3D

Final thoughts

It's not about Unity vs Web App, it's about your project: if it fits to Web or not.


If your app is a simple design with simple, less complicated features, go for a web app. If it’s a complicated application with sizeable graphics and animations or multiple motion events with WOW affect, then go for Unity.


The cost effectiveness of web apps makes it a better runner for non WoW affect apps but if you are looking for features that are out of the ordinary, Web App may not be the answer.
